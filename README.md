🚀 Orbital CRM

A high-performance Client Lead Management system designed to streamline sales pipelines and improve conversion efficiency.

📌 Overview

Orbital CRM is built to simplify how leads are managed—from initial contact to conversion. It provides a clear, real-time view of your pipeline while maintaining a fast, intuitive, and visually engaging user experience.

✨ Features
✅ Real-time Pipeline Tracking – Monitor leads from New to Converted at a glance
📊 Dynamic Dashboards – Get instant insights on conversion rates and lead sources
📝 Contextual Notes – Add timestamped notes to every lead for better tracking
🎨 Modern UI/UX – Clean, glassmorphic design for an engaging experience
📱 Fully Responsive – Works seamlessly across devices
🛠️ Tech Stack
Frontend: React
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Tokens)

⚙️ Key Highlights
🔹 Built with the MERN Stack for full-stack scalability
🔹 Secure authentication and protected routes
🔹 Optimized for performance and smooth user interaction
🔹 Component-based architecture for maintainability
🚀 Getting Started
1. Clone the repository
git clone htttps://github.com/kido19/FUTURE_FS_02
cd Orbital-CRM
2. Install dependencies
# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
3. Run the project
# Run backend
cd server
npm run dev

# Run frontend
cd ../client
npm start
